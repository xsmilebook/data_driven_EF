  - `state_*` 的 block 起止改为严格覆盖正式刺激序列，不再允许 `Trial_fix`、`Trial.started`、block cue 等更早时间进入 block regressors；
  - 保持 `response` 为实际按键时间 `key_resp.started + key_resp.rt`，并在代码中补充注释。
- 更新若干影像预处理 shell 入口，将 `python` 调用改为 `python3`，避免登录节点默认 Python 2.7 触发 `scripts.render_paths.py` 的语法错误：

- 文档中明确：
  - `MRI_Signal_s.started` 是开始等待 MRI 触发的时间；

  - `Begin_fix.started` 仅作为缺失时的回退近似值；
  - `key_resp.started` 是开始收集反应，不是实际按键时刻。
